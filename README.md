# B2C Platform - Version 1 (Angular)

This folder contains the Version 1 implementation of the B2C platform, built with Angular.

## 🚀 Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm start
   ```

3. **Open in Browser:**
   Navigate to `http://localhost:4200/`.

---

## 🏗 Key Features
- Angular 21 Components
- SCSS Styling
- Standard Angular Directory Structure
